# California Housing Price Prediction

A Machine Learning project that predicts California housing prices using the California Housing dataset from Scikit-Learn.

## Features

* Exploratory Data Analysis (EDA)
* Data Visualization
* Linear Regression
* Random Forest Regression
* Model Evaluation using MAE, MSE, RMSE, and R² Score

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Results

Compared Linear Regression and Random Forest models to predict housing prices based on demographic and geographic features. Random Forest achieved better prediction performance.
